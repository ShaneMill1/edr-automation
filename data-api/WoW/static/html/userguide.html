Example query tool user guide

Point Query

  1. Use the pan and zoom controls on the map to navigate to the area you are interested in

  2. Click on the icon below the map zoom controls, this will change the mouse icon to a location marker

  1. Click on the map location that you want to retreive data for.

  2. The Coordinates field on the left hand side should now be populated with a WKT string

  1. Click on the checkbox next to the parameter names for each parameter that you want to retreive values for.

  1. Change the values in the drop down lists below the parameter list to select the dimension ranges and output values that you want to retreive data for.

  1. Click the Get data button

  1. Once the query has finished running:

    1. If the query was for a time series a Chart of the returned values will be plotted.

  1. If the data returned a single value:

    1. Click on the checkbox for the parameter in the top right hand corner of the map.

    2. A blue dot will appear at the base of the location icon, click on the blue dot to display the data value

  1. At the bottom of the screen the URL generated by the choosen options will appear, click on the clipboard icon next to the URL to copy it.

Polygon Query

  1. Use the pan and zoom controls on the map to navigate to the area you are interested in

  2. Click on the polygon icon below the map zoom controls, this will change the mouse icon to a crosshair icon

  1. Click on the map for each vertices of the polygon (clicking on the first location to finish defining the polygon) to define the area to reterieve data for.

  1. The Coordinates field on the left hand side should now be populated with a WKT string

  1. Click on the checkbox next to the parameter names for each parameter that you want to retreive values for.

  1. Change the values in the drop down lists below the parameter list to select the dimension ranges and output values that you want to retreive data for.

  1. Click the Get data button

  1. When the query completes a checkbox will appear at the top right hand corner of the map, click to select the parameter that you want to view.

  1. If the data requested is multi dimensional (i.e. has height and time values) a control will appear on the left hand side of the map to change the time/level being viewed

  1. At the bottom of the screen the URL generated by the choosen options will appear, click on the clipboard icon next to the URL to copy it.
